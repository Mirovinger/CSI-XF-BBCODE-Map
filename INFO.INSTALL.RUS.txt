--------------------------------------------------------------------------------
Module Installation
--------------------------------------------------------------------------------

1. Загрузите все файлы из папки "upload", сохраняя структуру директорий.
2. Зайдите в панель управления, перейдите на вкладку "Главная" > Установить дополнение.
3. В параметре "Загрузить файл с компьютера" выберите XML-файл модуля.
4. Нажмите кнопку "Установить дополнение".
